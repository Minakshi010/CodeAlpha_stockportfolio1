# 📊 Stock Portfolio Tracker (GUI)

A Python GUI-based Stock Portfolio Tracker that allows users to manage their stock investments and generate reports.

## 🚀 Features

- Add stocks with quantity
- Calculate total investment
- Display portfolio in GUI
- Save report as CSV
- Save report as TXT
- Pie Chart visualization (Matplotlib)

## 🛠 Technologies Used

- Python
- Tkinter (GUI)
- Matplotlib
- CSV
- File Handling

## 🧠 Concepts Used

- Dictionary
- Loops
- Functions
- If-else
- File handling
- GUI programming
